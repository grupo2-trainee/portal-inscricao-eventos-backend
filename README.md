# 🎫 Portal de Inscrição de Eventos - Backend

Backend desenvolvido em **Node.js** com **Express**, responsável pela API de um sistema de gerenciamento de eventos, atividades e inscrições.


---

## 🧾 Descrição

Este é o backend do portal de inscrição de eventos do **Grupo 2 - Trainee**. A aplicação permite:

- Criar, listar e deletar eventos/atividades
- Realizar e listar inscrições
- Gerenciar capacidade de eventos/atividades

---

## ⚙️ Tecnologias

- Node.js
- Express
- Prisma
- PostgreSQL
- JWT
